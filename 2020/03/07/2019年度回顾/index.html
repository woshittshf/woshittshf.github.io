2019年：用 Anki 和 pomotodo 来提高学习效率

一切的开始

2019年我参加了一场340万人的研究生入学考试，为了能在考试中取得想要的成绩，我将几年来学习到的效率生活方式应用于备考当中，这篇文章将记录下我个人实践过后觉得有助于提高学习效率的工具和方法

知识点的记忆

既然是标准化考试，必然涉及到一定的知识点的记忆，作为一名在中学时期饱受语文、英语等记忆性课程摧残的理科生来说，如何快速高效的记忆知识点实在是一件至关重要的事情。

经过一段时间的搜索以后，我总结出了高效记忆知识点的几个有效的方法。

  1. 对于有理解量的知识点必须先理解，再进行记忆，若是英语单词这类理解量低的知识点则可以直接记忆
  2. 在完成第一步之后，使用 active recall 和 spaced repetition 这两个技巧来提高记忆的效率

active recall 主动召回

Active recall is a principle of efficient learning, which claims the need to actively stimulate memory during the learning process. It contrasts with passive review, in which the learning material is processed passively (e.g. by reading, watching, etc.). For example, reading a text about George Washington, with no further action, is a passive review. Answering the question “Who was the first US President?”, is active recall.

也就是说当我们记忆一个知识点的时候，尝试将知识点主动的从脑子里面提取出来对知识的记忆效果会远远好于直接把知识点塞进脑子里面

spaced repetition 空间重复

Spaced repetition is an evidence-based learning technique that is usually performed with flashcards. Newly introduced and more difficult flashcards are shown more frequently while older and less difficult flashcards are shown less frequently in order to exploit the psychological spacing effect. The use of spaced repetition has been shown to increase rate of learning.

简而言之就是当我们完成对一个知识点的记忆以后，在快要忘掉这个知识点的时候复习它，那么所获得的记忆效果才是最好的。
那么，什么时候才是“快要忘掉”的时候呢。一个参考的标准就是艾宾浩斯记忆曲线

根据艾宾浩斯记忆曲线所述，我们应该在记忆完一个知识点之后的1天、1周、及1个月的时间点复习该知识点

但是我们对于不同的知识点“快要忘掉”的时间点其实是不同的，如何确定该时间点和以及如何实现不同的知识点在不同的时间点记忆

我在搜索过程中找到了两种解决方案

  1. 制作实体的抽认卡，将不同熟悉程度的卡片放置在不同的格子当中，标记上时间，根据标记的时间进行复习，具体可以参考 thomas frank制作的 YouTube视频 以及其 博客文章

  2. 利用软件实现实体抽认卡的功能，甚至还能够拥有更多实体抽认卡所不具备的功能。此处我个人使用的软件是 Anki 这款软件

下面将以我以 Anki 这款软件进行单词的记忆为例阐述我的知识点记忆方式

概述

在2019年3月26日到2019年9月21日期间，我使用 Anki 这款软件记忆了6000+的单词，将自身的词汇量从 3000+ 提高到了 7000+

这给我带来的好处不仅仅是在标准化考试当中取得了一个令我相当满意的成绩，还使我对英文信息的吸收能力大大增强，感觉周围可吸收的有效信息一下子多了一个数量级，可以说获益匪浅

由于英语单词的性质，我并不需要个性化的 Anki 卡片，所以我直接使用了 赛门喵Simon 老师精心制作的 Anki 牌组

我的记忆方法是：

每天记忆50个新的单词+复习所有的背过的到期的单词

在背每一个单词的时候，都在脑子里面快速过一遍，然后确定对单词的熟悉程度，Anki 会根据你选择的熟悉程度决定你会在什么时候需要再次学习这张卡片

尽量避免间断，因为如果你间断一天的话，就会造成很大的损失，简单的计算如下：

让我们接着算一笔帐，假如你1天没背单词，你的时间损失有多大：首先，你损失了1天的进度；你次日复习的时候，会发现你上次（2天前）背的单词因为没有复习而忘光光，基本需要从头背起，学习曲线需要重置；你4天以前背的单词因为没有复习，记忆效率也打了五折；依次类推……所以，如果你60天里哪怕只偷懒了1天，你最终就需要63天才能背完所有单词。如果你每3天偷懒1天，你背单词的日子至少要多一倍！这还是假设你第二天把头一天的欠帐照单全吃下的情况。你正常情况下每天只需要复习500单词，因为你昨天没复习 ，你就需要复习900个单词。务必当天吃下，不然你还要付出代价。

更可怕的是，假如你本来复习计划需要你某月背完单词，你在绝望之下会发现自己为了完成计划，需要每天背200个新单词。这将转变成更大的压力：如果你每天背100个新词，大约会需要复习500个旧词；而你每天背200个新词，你会需要复习1000以上的旧词！如果你这时候还正好有一天偷懒没背，恭喜你，你第二天需要复习1800个单词以保证学习进度。而人的大脑是会疲劳的，背2小时单词的效率肯定要大大低于背1小时单词。如果这时候你望着1800需要复习的单词退却了，放弃复习了，好吧，像我上段里说的，你前功尽弃了。我很庆幸，我在犯过几次错误之后，即使面临着需要复习2000+单词的巨大压力，也把计划最终完成了。而很多人恐怕一旦看到需要复习2000+单词，计划就直接崩盘了（可能有同学会说，我如果有2000+单词要复习，我今天先搞定1500，明天再搞定1500，后天再搞定1500不行吗？可以的，但是你可以算一算，由于你没有遵照记忆曲线在该复习的时候复习，你仍然损失了大量背诵进度）

推荐在 iPad 端配合外接键盘来使用，iPad 端的软件优化非常好，响应速度比windows平台快很多，可以节省不少时间

最后，在经过了整整 6 个月的记忆以后，我完成了所有单词的记忆，达到了自己所设定的目标，收获很大

上述对 Anki 的用法只是一个简单的例子，其实在准备考试期间我还使用 Anki 做了很多事情，比如

  • 记忆政治简答题
  • 把数学错题制作成 Anki卡片，以达到强制自己完全搞懂一道题目的目的
  • 利用 Anki 制作《赖世雄美语音标》中的音标卡片，帮助自己在学完该书后巩固记忆

事实上，Anki 还有很多强大的功能，也还有许许多多的用法，Anki 官方网站中就能找到各种领域的 卡片
如果你有什么需要记忆的知识的话完全可以开始尝试使用

利用番茄工作法来降低学习的阻力，增加学习的动力

在准备考试期间，经常一整天坐在桌子前面学习，作为一名“意志力”不太坚强的人，经常觉得撑不下来。再加上一整天坐着长时间下来身体也会吃不消，正确的做法应该是每隔一段时间就站起来活动活动，既可以缓解颈部和腰部的疲劳，又可以放松一下大脑，提高之后时间的学习效率。
为了解决这一个问题，我采用了称为番茄工作法的方法。

What

番茄工作法：Wikipedia 条目
就是指在开始工作的时候设定一个时间段长度，在该时间段结束的时候休息，同样设定休息的时间长度，休息完成后接着工作，周而复始
似乎看起来没什么用处，然而我在实践了一段时间以后发现这个工作法的效果是很大的

Why

我们面对一个”任务“的时候，传统的 GTD 的做法是将其分切为一个个”下一步行动“，避免我们在面对一个庞大的任务是因为迷茫而造成的拖延
但是，当我们使用番茄工作法的时候，这种状况就不会出现，因为此时我们的心理状况会是这样：虽然这个任务看起来很复杂，毫无头绪，但是不管是什么样的任务，我们去解决它所花费的时间都不会改变，都得消耗一个一个番茄的时间去解决问题，那我有什么可以犹豫的呢？开始一个番茄不就行了
这个思维方式尤其适合在我们难以（懒得）将任务划分为下一步行动时，直接开始一个番茄，之后的事情再说

当我们开启一个时段的学习的时候，如果我们脑子里面想的是我接下来要学习的时间是3个小时，很有可能还没开始学就产生了畏难情绪：我能够学到3个小时吗？
当当我们使用番茄的工作法的时候，就会这么想：3小时，也只是6个番茄而已，先开始学再说，哪怕学不到6个番茄，学4个、2个也是好的吗，而一旦你开始了学习之后，往往就很难停下来，基本都会学满整个3小时

此外还有一些理由，比如说人的大脑并不是为了长时间专注而设定的，采取工作/休息交替的方式往往能够使得效率最大化

how

具体对应到实际操作上面，我在 windows 端和 iOS 端用的 app 都是 pomotodo 这款 app，它有以下几个优点

  • 全平台，覆盖主流所有平台，包括windows，Mac OS，iOS，android，web等等
  • 高级版可以将完成的番茄钟订阅到日历中，一目了然
  • 够用的数据分析系统，可以选择对应的时间段分析数据，对于回顾十分友好

尾巴

在 2019 年度的标准化考试准备过程中，以 Anki 为核心的记忆体系和以 pomotodo 为核心的任务管理体系帮了我很多，最终我也取得了想取得的成绩，希望这两个系统也能够给你带来一定的启发。
